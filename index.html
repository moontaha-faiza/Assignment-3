CSS Grid and Flexbox are layout models that are similar and can be used in combination. The main difference is that CSS Grid can build two-dimensional layouts, whereas Flexbox can only produce one-dimensional layouts. That is, in CSS Grid, you can put components along both the X and Y axes, whereas Flexbox only supports one axis.

Bootstrap is a Component-based Framework. It's intended to assist developers quickly bootstrap a standard website or app, therefore it includes a set of pre-made components as well as various utilities for spacing and so on. On the other hand,TailwindCSS is a framework focused on utility. Unlike Bootstrap, which provides a few conveniences, Tailwind is entirely utilitarian. Because there are no pre-made components in the core framework, you must create your own.

HTML semantics refers to the tags in an HTML page that provide meaning rather than just display. It clarifies HTML by better describing the various sections and layout of web pages. By including semantic tags in any work, you provide more information about it, which enhances communication. Semantic tags, in particular, inform browsers about the meaning of a page and its content.

The word "box model" is used in CSS when discussing design and layout. Every HTML element is essentially enclosed in a box thanks to the CSS box model. It consists of the content itself, as well as margins, borders, and padding. Block boxes and inline boxes are the two main forms of boxes found in CSS. The type describes how the box interacts with other boxes on the page and with the flow of the page. There are two types of displays for boxes: inner and outer.
